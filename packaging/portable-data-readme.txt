VRCTranslate 的配置、日志、用户术语、OCR/语音模型和下载缓存会保存在本 data 目录。
用户术语位于 glossaries\user_glossary.json，只在本机软件目录中使用。
正式 OCR 模型位于 models\ocr；cache\ocr-models 只保存下载临时文件并会自动清理。
本地 SenseVoice 位于 models\speech 和 components\local-asr；麦克风音频、识别原文与译文不会保存到 data。
请勿单独移动 exe；应移动或备份整个 VRCTranslate 文件夹。
若不再需要软件，删除整个软件文件夹即可移除这些数据。
